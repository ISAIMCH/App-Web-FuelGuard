## PASO 1️⃣: Abrir Firewall en Google Cloud (2 minutos)

1. Ir a: https://console.cloud.google.com/
2. Menú → **VPC Network** → **Firewall rules**
3. Click: **Create Firewall Rule**
4. Configurar:
   ```
   Name: allow-sensorhub
   Direction: Ingress
   Action: Allow
   Protocol: TCP
   Port: 8000
   Source IPv4: 0.0.0.0/0
   ```
5. Click: **Create**

✅ **Listo**

---

## PASO 2️⃣: Conectar por SSH (1 minuto)

**Opción A: Desde Google Cloud Console**
- Ir a Compute Engine → Instances
- Click en: **SSH in browser**

**Opción B: Desde terminal local**
```bash
gcloud compute ssh tu-instancia --zone=us-central1-a
```

---

## PASO 3️⃣: Ejecutar Script de Instalación (5 minutos)

En la terminal de Google Cloud:

```bash
# Ir a tu carpeta del proyecto
cd ~/AplicacionWeb

# Hacer script ejecutable
chmod +x google-cloud-install.sh

# Ejecutar instalación automática
./google-cloud-install.sh
```

**El script hace todo:**
✅ Instala Node.js
✅ Instala dependencias
✅ Configura PM2
✅ Inicia servidor automáticamente

## 📊 Comandos Útiles en Google Cloud

```bash
# Ver estado del servidor
pm2 status

# Ver logs en tiempo real
pm2 logs sensorhub

# Reiniciar servidor
pm2 restart sensorhub

# Detener servidor
pm2 stop sensorhub

# Iniciar de nuevo
pm2 start sensorhub
```

## ❌ Troubleshooting

### "No puedo conectar a http://34.145.56.208:8000"
```
1. Verificar firewall abierto (Paso 1)
2. Ver logs: pm2 logs sensorhub
3. Reiniciar: pm2 restart sensorhub
```

### "Error: port 8000 in use"
```
pm2 stop sensorhub
pm2 delete sensorhub
./google-cloud-install.sh
```

### "No hay datos en dashboard"
```
Normal si no tienes LilyGO enviando datos aún.
Espera a conectar el dispositivo.
```
