⚡

Recepción y visualización instantánea de datos mediante protocolo MQTT de bajo overhead

🔐

Autenticación Segura

Sistema de login con encriptación JWT y almacenamiento seguro de credenciales

📊

Análisis Histórico

📍

Geolocalización

Integración de GPS del SIM7070 para rastreo en tiempo real

Almacenamiento Local

Buffer en dispositivo para sincronización cuando hay pérdida de conexión

Especificaciones Técnicas

Hardware

  • Dispositivo IoT: LilyGO SIM7070
  • Conectividad: 4G/LTE, GPS integrado
  • Comunicación: ESP-NOW + MQTT
  • Sensors: Temperatura, Humedad, Presión
  • Precisión GPS: ±5 metros

Backend

  • Servidor: Node.js + Express
  • Broker MQTT: HiveMQ Cloud
  • Base de Datos: SQLite
  • Autenticación: JWT + bcrypt
  • API REST: Endpoints asegurados

Frontend

  • Visualización: Charts.js
  • Actualización: Polling cada 5 segundos
  • Responsive: Móvil, Tablet, Desktop
  • Interactividad: Gráficos dinámicos
  • Rango: Última 24h e histórico 7 días
